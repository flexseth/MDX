# RadioControl Component

`RadioControl` renders a group of radio buttons where only one option can be selected at a time. Use it when the choices are mutually exclusive. For multiple independent toggles use `CheckboxControl`; for a simple dropdown use `SelectControl`.

## Installation

This component is included in the `@wordpress/components` package, which is already available when using `@wordpress/create-block`.

## Required Imports

```javascript
import { RadioControl } from '@wordpress/components';
```

## block.json Attributes

```json
{
	"attributes": {
		"displayStyle": {
			"type": "string",
			"default": "list"
		}
	}
}
```

## Basic Usage in edit.js

```javascript
import { __ } from '@wordpress/i18n';
import { useBlockProps, InspectorControls } from '@wordpress/block-editor';
import { PanelBody, RadioControl } from '@wordpress/components';

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const { displayStyle } = attributes;

	return (
		<>
			<InspectorControls>
				<PanelBody title={ __( 'Display', 'your-text-domain' ) }>
					<RadioControl
						label={ __( 'Display style', 'your-text-domain' ) }
						selected={ displayStyle }
						options={ [
							{ label: __( 'List', 'your-text-domain' ), value: 'list' },
							{ label: __( 'Grid', 'your-text-domain' ), value: 'grid' },
							{ label: __( 'Carousel', 'your-text-domain' ), value: 'carousel' },
						] }
						onChange={ ( value ) => setAttributes( { displayStyle: value } ) }
					/>
				</PanelBody>
			</InspectorControls>
			<div { ...blockProps } className={ `display-${ displayStyle }` }>
				{ __( 'Content displayed as:', 'your-text-domain' ) } { displayStyle }
			</div>
		</>
	);
}
```

## Layout Mode Selection

```javascript
import { __ } from '@wordpress/i18n';
import { useBlockProps, InspectorControls } from '@wordpress/block-editor';
import { PanelBody, RadioControl } from '@wordpress/components';

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const { layout } = attributes;

	const layoutStyles = {
		stacked: { display: 'flex', flexDirection: 'column', gap: '16px' },
		side: { display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '16px' },
		full: { display: 'block' },
	};

	return (
		<>
			<InspectorControls>
				<PanelBody title={ __( 'Layout', 'your-text-domain' ) }>
					<RadioControl
						label={ __( 'Image layout', 'your-text-domain' ) }
						help={ __( 'Choose how the image is positioned relative to the text.', 'your-text-domain' ) }
						selected={ layout }
						options={ [
							{ label: __( 'Stacked (image above text)', 'your-text-domain' ), value: 'stacked' },
							{ label: __( 'Side by side', 'your-text-domain' ), value: 'side' },
							{ label: __( 'Full width image', 'your-text-domain' ), value: 'full' },
						] }
						onChange={ ( value ) => setAttributes( { layout: value } ) }
					/>
				</PanelBody>
			</InspectorControls>
			<div { ...blockProps } style={ layoutStyles[ layout ] }>
				<div className="image-area">{ __( '[Image]', 'your-text-domain' ) }</div>
				<div className="text-area">{ __( '[Text]', 'your-text-domain' ) }</div>
			</div>
		</>
	);
}
```

## Button Size Selection

```javascript
import { __ } from '@wordpress/i18n';
import { useBlockProps, InspectorControls } from '@wordpress/block-editor';
import { PanelBody, RadioControl } from '@wordpress/components';

const SIZE_MAP = {
	small: { padding: '6px 12px', fontSize: '12px' },
	medium: { padding: '10px 20px', fontSize: '16px' },
	large: { padding: '14px 28px', fontSize: '20px' },
};

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const { buttonSize, buttonLabel } = attributes;

	return (
		<>
			<InspectorControls>
				<PanelBody title={ __( 'Button Settings', 'your-text-domain' ) }>
					<RadioControl
						label={ __( 'Button size', 'your-text-domain' ) }
						selected={ buttonSize }
						options={ [
							{ label: __( 'Small', 'your-text-domain' ), value: 'small' },
							{ label: __( 'Medium', 'your-text-domain' ), value: 'medium' },
							{ label: __( 'Large', 'your-text-domain' ), value: 'large' },
						] }
						onChange={ ( value ) => setAttributes( { buttonSize: value } ) }
					/>
				</PanelBody>
			</InspectorControls>
			<div { ...blockProps }>
				<button style={ SIZE_MAP[ buttonSize ] }>
					{ buttonLabel || __( 'Click me', 'your-text-domain' ) }
				</button>
			</div>
		</>
	);
}
```

## Dynamic Options from State

```javascript
import { __ } from '@wordpress/i18n';
import { useBlockProps, InspectorControls } from '@wordpress/block-editor';
import { PanelBody, RadioControl, Spinner } from '@wordpress/components';
import { useState, useEffect } from '@wordpress/element';
import apiFetch from '@wordpress/api-fetch';

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const { selectedCategory } = attributes;
	const [ categories, setCategories ] = useState( [] );
	const [ isLoading, setIsLoading ] = useState( true );

	useEffect( () => {
		apiFetch( { path: '/wp/v2/categories?per_page=10' } )
			.then( ( data ) => {
				setCategories(
					data.map( ( cat ) => ( { label: cat.name, value: String( cat.id ) } ) )
				);
				setIsLoading( false );
			} )
			.catch( () => setIsLoading( false ) );
	}, [] );

	return (
		<>
			<InspectorControls>
				<PanelBody title={ __( 'Category', 'your-text-domain' ) }>
					{ isLoading ? (
						<Spinner />
					) : (
						<RadioControl
							label={ __( 'Filter by category', 'your-text-domain' ) }
							selected={ selectedCategory }
							options={ [
								{ label: __( 'All categories', 'your-text-domain' ), value: '' },
								...categories,
							] }
							onChange={ ( value ) => setAttributes( { selectedCategory: value } ) }
						/>
					) }
				</PanelBody>
			</InspectorControls>
			<div { ...blockProps }>
				{ selectedCategory
					? `${ __( 'Showing category ID:', 'your-text-domain' ) } ${ selectedCategory }`
					: __( 'Showing all posts', 'your-text-domain' )
				}
			</div>
		</>
	);
}
```

### block.json for dynamic options

```json
{

		"selectedCategory": {
			"type": "string",
			"default": ""
		}
	}
}
```

## save.js

`RadioControl` is a sidebar control; its selected value is used in the save output:

```javascript
import { useBlockProps } from '@wordpress/block-editor';

export default function save( { attributes } ) {
	const blockProps = useBlockProps.save();
	const { displayStyle } = attributes;

	return (
		<div { ...blockProps } className={ `display-${ displayStyle }` }>
			{ /* Render content based on displayStyle */ }
		</div>
	);
}
```

## Props Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | string | - | Group label displayed above the radio buttons |
| `selected` | string | - | Value of the currently selected option |
| `options` | array | `[]` | Array of `{ label, value }` objects |
| `onChange` | function | - | Callback receiving the newly selected value string |
| `help` | string\|node | - | Descriptive text below the group |
| `disabled` | boolean | `false` | Disables all radio buttons |
| `className` | string | - | Additional CSS classes on the wrapper |
| `hideLabelFromVision` | boolean | `false` | Visually hides the group label (still read by screen readers) |

## Common Use Cases

1. **Display styles**: List, grid, masonry, carousel
2. **Layout modes**: Stacked, side-by-side, full width
3. **Size variants**: Small, medium, large
4. **Alignment options**: When only one alignment is valid at a time
5. **Content source**: Latest posts, specific category, custom query

## Accessibility

- Renders as a `<fieldset>` with a `<legend>` for the group label
- Individual radio inputs are `<input type="radio">` with associated `<label>` elements
- Keyboard navigation: arrow keys move between options within the group
- `help` text is linked via `aria-describedby`

## WordPress Standards

- Import from `@wordpress/components`
- Use `@wordpress/i18n` for all `label`, `options.label`, and `help` strings
- `options` values should be strings even when representing numbers (`value: '3'` not `value: 3`)
- For fewer than 4 options, `RadioControl` is often clearer than `SelectControl`
- For more than 6 options, switch to `SelectControl` to avoid visual clutter
