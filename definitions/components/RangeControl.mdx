# RangeControl Component

The `RangeControl` component provides a slider input for selecting numeric values within a specified range.

## Installation

This component is included in the `@wordpress/components` package, which is already available when using `@wordpress/create-block`.

## Required Imports

```javascript
import { RangeControl } from '@wordpress/components';
```

## block.json Attributes

```json
{
	"attributes": {
		"fontSize": {
			"type": "number",
			"default": 16
		}
	}
}
```

## Basic Usage in edit.js

```javascript
import { __ } from '@wordpress/i18n';
import { RangeControl } from '@wordpress/components';
import { useBlockProps } from '@wordpress/block-editor';

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const { fontSize } = attributes;

	return (
		<div { ...blockProps }>
			<RangeControl
				label={ __( 'Font Size', 'your-text-domain' ) }
				value={ fontSize }
				onChange={ ( value ) => setAttributes( { fontSize: value } ) }
				min={ 12 }
				max={ 48 }
			/>
			<p style={ { fontSize: `${ fontSize }px` } }>
				{ __( 'Sample text with custom font size', 'your-text-domain' ) }
			</p>
		</div>
	);
}
```

## Props Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | string | - | Label text for the range control |
| `value` | number | - | Current value of the slider |
| `onChange` | function | - | Callback when value changes |
| `min` | number | `0` | Minimum value |
| `max` | number | `100` | Maximum value |
| `step` | number | `1` | Step increment |
| `help` | string | - | Help text displayed below the control |
| `marks` | array | - | Array of marks/tick marks to display |
| `disabled` | boolean | `false` | Disable the control |
| `allowReset` | boolean | `false` | Show reset button |
| `resetFallbackValue` | number | - | Value to use when reset is clicked |
| `className` | string | - | Additional CSS classes |
| `withInputField` | boolean | `true` | Show/hide the numeric input field |

## Common Use Cases

1. **Font Sizing**: Control text size with min/max bounds
2. **Spacing/Padding**: Adjust margins, padding, gaps
3. **Opacity**: Control transparency (0-1 with step 0.1)
4. **Columns/Grid**: Set number of columns or grid items
5. **Border Properties**: Control border radius, width

## Best Practices

- Set appropriate `min` and `max` values
- Use `step` for precision control (e.g., 0.1 for decimals)
- Provide `help` text for complex controls
- Set sensible defaults in `block.json`
- Consider using `marks` for discrete values
- Show live preview of the value's effect

## Accessibility

- Automatically labeled for screen readers
- Keyboard navigable (arrow keys)
- Supports focus management
- Works with assistive technologies

## WordPress Standards

- Import from `@wordpress/components`
- Use `@wordpress/i18n` for translatable text
- Define numeric attributes in `block.json`
- Use number type for attributes
