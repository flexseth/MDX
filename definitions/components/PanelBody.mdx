# PanelBody Component

The `PanelBody` component provides a collapsible container for grouping related settings, commonly used in the block inspector sidebar.

## Installation

This component is included in the `@wordpress/components` package, which is already available when using `@wordpress/create-block`.

## Required Imports

```javascript
import { PanelBody, PanelRow } from '@wordpress/components';
import { InspectorControls } from '@wordpress/block-editor';
```

## Basic Usage in edit.js

```javascript
import { __ } from '@wordpress/i18n';
import { PanelBody, TextControl } from '@wordpress/components';
import { useBlockProps, InspectorControls } from '@wordpress/block-editor';

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const { title } = attributes;

	return (
		<>
			<InspectorControls>
				<PanelBody title={ __( 'Settings', 'your-text-domain' ) }>
					<TextControl
						label={ __( 'Title', 'your-text-domain' ) }
						value={ title }
						onChange={ ( value ) => setAttributes( { title: value } ) }
					/>
				</PanelBody>
			</InspectorControls>
			<div { ...blockProps }>
				<h2>{ title || __( 'Add title...', 'your-text-domain' ) }</h2>
			</div>
		</>
	);
}
```

		}
	}
}

## Multiple PanelBody Components

```javascript
import { __ } from '@wordpress/i18n';
import {
	PanelBody,
	TextControl,
	ToggleControl,
	RangeControl,
} from '@wordpress/components';
import { useBlockProps, InspectorControls } from '@wordpress/block-editor';

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const { heading, showDate, fontSize } = attributes;

	return (
		<>
			<InspectorControls>
				<PanelBody
					title={ __( 'Content Settings', 'your-text-domain' ) }
					initialOpen={ true }
				>
					<TextControl
						label={ __( 'Heading', 'your-text-domain' ) }
						value={ heading }
						onChange={ ( value ) => setAttributes( { heading: value } ) }
					/>
					<ToggleControl
						label={ __( 'Show Date', 'your-text-domain' ) }
						checked={ showDate }
						onChange={ ( value ) => setAttributes( { showDate: value } ) }
					/>
				</PanelBody>

				<PanelBody
					title={ __( 'Typography Settings', 'your-text-domain' ) }
					initialOpen={ false }
				>
					<RangeControl
						label={ __( 'Font Size', 'your-text-domain' ) }
						value={ fontSize }
						onChange={ ( value ) => setAttributes( { fontSize: value } ) }
						min={ 12 }
						max={ 72 }
					/>
				</PanelBody>
			</InspectorControls>
			<div { ...blockProps }>
				<h2 style={ { fontSize: `${ fontSize }px` } }>
					{ heading || __( 'Add heading...', 'your-text-domain' ) }
				</h2>
				{ showDate && <p>{ new Date().toLocaleDateString() }</p> }
			</div>
		</>
	);
}
```

## Using PanelRow

```javascript
import { __ } from '@wordpress/i18n';
import { PanelBody, PanelRow, Button, TextControl } from '@wordpress/components';
import { useBlockProps, InspectorControls } from '@wordpress/block-editor';

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const { url } = attributes;

	return (
		<>
			<InspectorControls>
				<PanelBody title={ __( 'Link Settings', 'your-text-domain' ) }>
					<PanelRow>
						<TextControl
							label={ __( 'URL', 'your-text-domain' ) }
							value={ url }
							onChange={ ( value ) => setAttributes( { url: value } ) }
						/>
					</PanelRow>
					<PanelRow>
						<Button
							variant="secondary"
							onClick={ () => setAttributes( { url: '' } ) }
						>
							{ __( 'Clear URL', 'your-text-domain' ) }
						</Button>
					</PanelRow>
				</PanelBody>
			</InspectorControls>
			<div { ...blockProps }>
				{ url && <a href={ url }>{ __( 'Link', 'your-text-domain' ) }</a> }
			</div>
		</>
	);
}
```

## Controlled Open/Close State

```javascript
import { __ } from '@wordpress/i18n';
import { PanelBody, TextControl } from '@wordpress/components';
import { useBlockProps, InspectorControls } from '@wordpress/block-editor';
import { useState } from '@wordpress/element';

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const { content } = attributes;
	const [ isPanelOpen, setIsPanelOpen ] = useState( true );

	return (
		<>
			<InspectorControls>
				<PanelBody
					title={ __( 'Content', 'your-text-domain' ) }
					opened={ isPanelOpen }
					onToggle={ () => setIsPanelOpen( ! isPanelOpen ) }
				>
					<TextControl
						label={ __( 'Content', 'your-text-domain' ) }
						value={ content }
						onChange={ ( value ) => setAttributes( { content: value } ) }
					/>
				</PanelBody>
			</InspectorControls>
			<div { ...blockProps }>
				<p>{ content }</p>
			</div>
		</>
	);
}
```

		"content": {
			"type": "string",

		}
	}
}

## Comprehensive Settings Example

```javascript
import { __ } from '@wordpress/i18n';
import {
	PanelBody,
	PanelRow,
	TextControl,
	ToggleControl,
	RangeControl,
	SelectControl,
	ColorPalette,
} from '@wordpress/components';
import { useBlockProps, InspectorControls } from '@wordpress/block-editor';

export default function Edit( { attributes, setAttributes } ) {
	const blockProps = useBlockProps();
	const {
		title,
		subtitle,
		showImage,
		columns,
		alignment,
		backgroundColor,
	} = attributes;

	const colors = [
		{ name: 'Red', color: '#f44336' },
		{ name: 'Blue', color: '#2196f3' },
		{ name: 'Green', color: '#4caf50' },
	];

	return (
		<>
			<InspectorControls>
				<PanelBody
					title={ __( 'Content', 'your-text-domain' ) }
					initialOpen={ true }
				>
					<TextControl
						label={ __( 'Title', 'your-text-domain' ) }
						value={ title }
						onChange={ ( value ) => setAttributes( { title: value } ) }
					/>
					<TextControl
						label={ __( 'Subtitle', 'your-text-domain' ) }
						value={ subtitle }
						onChange={ ( value ) => setAttributes( { subtitle: value } ) }
					/>
					<ToggleControl
						label={ __( 'Show Featured Image', 'your-text-domain' ) }
						checked={ showImage }
						onChange={ ( value ) => setAttributes( { showImage: value } ) }
					/>
				</PanelBody>

				<PanelBody
					title={ __( 'Layout', 'your-text-domain' ) }
					initialOpen={ false }
				>
					<RangeControl
						label={ __( 'Columns', 'your-text-domain' ) }
						value={ columns }
						onChange={ ( value ) => setAttributes( { columns: value } ) }
						min={ 1 }
						max={ 4 }
					/>
					<SelectControl
						label={ __( 'Alignment', 'your-text-domain' ) }
						value={ alignment }
						options={ [
							{ label: __( 'Left', 'your-text-domain' ), value: 'left' },
							{ label: __( 'Center', 'your-text-domain' ), value: 'center' },
							{ label: __( 'Right', 'your-text-domain' ), value: 'right' },
						] }
						onChange={ ( value ) => setAttributes( { alignment: value } ) }
					/>
				</PanelBody>

				<PanelBody
					title={ __( 'Colors', 'your-text-domain' ) }
					initialOpen={ false }
				>
					<PanelRow>
						<label>{ __( 'Background Color', 'your-text-domain' ) }</label>
					</PanelRow>
					<ColorPalette
						colors={ colors }
						value={ backgroundColor }
						onChange={ ( color ) => setAttributes( { backgroundColor: color } ) }
					/>
				</PanelBody>
			</InspectorControls>

			<div
				{ ...blockProps }
				style={ {
					backgroundColor: backgroundColor,
					textAlign: alignment,
					padding: '20px',
				} }
			>
				{ title && <h2>{ title }</h2> }
				{ subtitle && <p>{ subtitle }</p> }
			</div>
		</>
	);
}
```

## Props Reference

### PanelBody

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | string | - | Panel header title |
| `icon` | Component | - | Icon component from `@wordpress/icons` |
| `initialOpen` | boolean | `true` | Whether panel is open by default |
| `opened` | boolean | - | Controlled open state |
| `onToggle` | function | - | Callback when panel is toggled |
| `className` | string | - | Additional CSS classes |
| `children` | node | - | Panel content |

### PanelRow

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | string | - | Additional CSS classes |
| `children` | node | - | Row content |

## Common Use Cases

1. **Settings Organization**: Group related settings together
2. **Inspector Controls**: Standard sidebar settings panel
3. **Progressive Disclosure**: Hide advanced settings by default
4. **Visual Hierarchy**: Organize complex interfaces
5. **Collapsible Sections**: Save space in crowded sidebars

## Best Practices

- Use descriptive panel titles
- Set `initialOpen={ true }` for most important panels
- Group logically related controls together
- Don't nest PanelBody components
- Use PanelRow for custom layouts within panels
- Keep panels focused on a single topic
- Order panels by importance (most used first)

## Accessibility

- Keyboard navigable (Enter/Space to toggle)
- ARIA attributes handled automatically
- Focus management built-in
- Screen reader friendly
- Supports assistive technologies
